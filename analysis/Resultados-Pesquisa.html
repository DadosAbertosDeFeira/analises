Inicialmente, podemos ver que todos os campos do resultado da pesquisa consistem de objetos, provavelmente representando informaes textuais. Isso tambm resultado da forma como os dados foram armazenados na planilha no Google Sheets.

Analisando as colunas, podemos considerar que a informao de data/hora da resposta da pesquisa pode ser descartada, j que no traz um dado que possa ser analisado em conjunto com os demais, representando apenas o momento em que aquela resposta foi preenchida.

A maioria das demais colunas possuem quase que todos os registros com valores nicos. Supondo que isso seja resultado da insero da informao por um campo de texto, podemos tentar analisar respostas que sejam semelhantes e agrup-las, criando categorias que possam ser mais teis para a anlise.

A coluna que menos apresenta valores nicos para os seus registros a pergunta "Com qual frequncia voc busca por informaes fornecidas pela prefeitura de Feira de Santana?". Supondo que a pergunta tenha disponibilizado opes para resposta, isso faz sentido. Ainda assim, a pergunta supe uma ordenao entre as respostas, que buscaremos evidenciar.

def categorize_profession(profession):
    if 'militar' in profession.lower():
        return 'Militar'
    if 'professor' in profession.lower():
        return 'Professor/Professora'
    if profession in ('Promotor De Vendas', 'Representante'):
        return 'Vendedor/Representante'
    if 'publicit' in profession.lower():
        return 'Publicitrio/Publicitria'
    if 'community manager' in profession.lower():
        return 'Organizador de Comunidade'
    else:
        return profession

pesquisa['OCUPACAO_CATEGORY'] = pesquisa['OCUPACAO'].apply(categorize_profession)

Assim como fizemos em outras perguntas, podemos estabelecer categorias atravs de palavras-chave. Como a quantidade de respostas no muito extensa, possvel fazer uma anlise em cada uma das respostas.

Em uma observao das respostas, podemos estabelecer as seguintes categorias:

  1. Investimentos (infraestrutura, despesas gerais, salrio);
  2. Sade;
  3. Educao;
  4. Segurana;
  5. Oramento (verba);
  6. Transparncia;
  7. Outros;

Uma resposta pode estar representada em mais de uma categoria. Assim, a soma das categorias pode ser maior que a quantidade de respostas obtidas.
